# Karo Arthropod

Karo Arthropod is a 1.20.1 Mohist survival server built around the terrifying and immersive "Arthropod Phobia Expansions" mod.

Face evolved insects.  
Survive brutal encounters.  
Explore a world where the smallest creatures are the greatest threat.

Custom features, balanced progression, and a challenging survival experience await.

If you're not afraid of what crawls in the dark... Join us.

---

## Modpack Information

- Minecraft Version: 1.20.1
- Server Software: Mohist
- Main Mod: Arthropod Phobia Expansions
- Multiplayer Survival Focused

---

## How To Join

### 1) Download & Install CurseForge
Download CurseForge from:
https://www.curseforge.com/download

---

### 2) Install The Modpack
- Download the .zip file from this repository
- Open CurseForge
- Click "Create Custom Profile"
- Click "Import"
- Select the downloaded .zip file

---

### 3) Launch & Connect
- Hit Play
- Navigate to Multiplayer
- Click Add Server
- Paste the domain below:

karoarthropod.my.pebble.host
